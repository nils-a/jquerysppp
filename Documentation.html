In order to the JQuery SharePoint people picker you first need to reference the JQUery UI bundle which can be found here: http://jqueryui.com/demos/   

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <title></title>
    <script src="js/jquery-1.7.2.min.js" type="text/javascript"></script>
    <script src="js/jquery-ui-1.8.22.custom.min.js" type="text/javascript"></script>
    <link href="css/cupertino/jquery-ui-1.8.22.custom.css" rel="stylesheet" type="text/css" />
    <script src="js/jquery-ui-sppeoplepicker-1.0.js" type="text/javascript"></script>
    <script>
        $(function () {

            $("#spp").peoplepicker({ server: 'http://myservername' });

            $("#testbutton").button().click(function () {
                alert($("#spp").val());
            });
        });
    </script>
</head>
<body>
    <div id="spp">
    </div>
    <button id="testbutton">text</button>  
</body>
</html>

Where server name is the url to your sharepoint server. The control will use _vti_bin/search.asmx to get the list of options for the drop down list. (So you must have a people search scope)

*Current scope name is "People", next version will allow to change it via the options.
